# FizzBuzz en Java

Este es un simple programa FizzBuzz escrito en Java. Muestra los números del 1 al 100 con las siguientes reglas:

- Si el número es divisible por 3, se imprime "FIZZ".
- Si el número es divisible por 5, se imprime "BUZZ".
- Si el número es divisible tanto por 3 como por 5, se imprime "FIZZ_BUZZ".
- En cualquier otro caso, se imprime el número.
